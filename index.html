jQTipnav

A jQuery plugin which enables showing small navigation bars on interaction with action element.

Usage: 
$element.tipnav(options);

where $element is jquery object pointing to unordered list elements - 'ul' tags. Options must contain 'action_element' jquery object. Please visit github for more detailed usage and contribution instructions.

jQTipnav applied to single element: 

$('ul#subnav').tipnav({ 'action_el': 'a#link1' });

hover to show

jQTipnav applied to multiple elements: 

$('ul.subnav').tipnav({ 'action_el': 'a#link1' });

hover to show

Additional options:
fade_time ->
fade out time in milliseconds
trigger_event ->
use 'click' event to trigger navigation bar visibility
