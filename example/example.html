TipNav usage

TipNav is small JQuery plugin that enables implementing small navigation lists showing when mouse hovers over or clicks on action element.

Usage: 

$element.tipnav(options);

Options argument must contain 'action_element' jquery object.

Tipnav applied to single element. 

$('ul#subnav').tipnav({ 'action_el': 'a#link1' });

hover to show

Tipnav applied to multiple elements. 

$('ul.subnav').tipnav({ 'action_el': 'a#link1' });

hover to show

Additional options:
fade_time ->
fade out time in milliseconds
trigger_event ->
use 'click' to trigger tipnav
